Why VLANs are needed.

- I created the vlans so that the network can be logically segmented into sep
 Broadcast domains, improving security, performances, and network management. So plans separated the departments like HR, Finance and IT into different network even if they are on the same physical switch

Why I used inter vlans
- implemented to allow communication between different departments (VLANs) while maintaining network segmentation. This was achieved using Router-on-a-Stick, where a single router interface handles multiple VLANs using subinterfaces.

Trunk links were configured to allow multiple VLAN traffic between switches and the router using IEEE 802.1Q tagging.
